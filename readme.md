Було реалізовано метод Монте Карло для розрахунку інтеграла. Отримані результати збігаються  
з розрахованими аналітичним методом, а точність розрахунук збільшується при збільшенні виборки  
випадкових величин.  
Хоча, продемонстрований приклад  не має практичного застосування, він демонструє простоту та  
можливості використання метода Монте Карло для розв'язку різних задач.
